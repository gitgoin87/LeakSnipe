LeakSnipe

Multi-site poker hand tracker with AI-assisted analysis — HUD, trainer, and web tools in one repo.

Features How it works Get started

Collect hands from multiple sites — analyze with AI

LeakSnipe imports raw hand histories from supported sites, stores them in a thread-safe SQLite database, runs analytics to surface leaks (VPIP, PFR, C-bet, AF), and provides a live HUD overlay and training tools to fix them.

Get started

  1. Clone the repo and open the README for per-subproject instructions
  2. Run the tracker: python poker_gui.py (requires customtkinter, pywin32)
  3. Run web trainer: cd Poker-Suite && flask run
  4. Build trainer: cd PokerBuild/poker-trainer && npm install && npm run electron
Want it live? Enable GitHub Pages from this repository's settings and serve from the docs/ folder.

How LeakSnipe works

  • Import: File watchers or manual import read hand-history files and store canonical Hand objects in SQLite.
  • Analyze: LeakEngine computes per-player stats and suggests fixes; AI models flag non-standard lines.
  • Live: CurrentHandMonitor drives the HUD overlay to show seat-level badges with alerts.
  • Train: Replay hands and use the web therapist to practice exploit fixes and GTO alternatives.
